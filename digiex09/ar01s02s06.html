2.6. Schedule

 
SEOC will make a long MFSK transmission prior to 9 o'clock to help stations to set up. Between scheduled transmissions SEOC will transmit in the upcoming mode to keep the frequency clear and allow stations a chance to set up for the next mode. Should a station be unable to start on time, the schedule will be slid 5 minutes.
